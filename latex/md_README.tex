 {\bfseries{Cub3D}} is a {\bfseries{Wolfenstein 3D-\/inspired}} raycasting project developed as part of the 42 curriculum. It provides a {\bfseries{basic 3D rendering engine}} using {\bfseries{raycasting}}\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Raycasting Engine}}\+: Implements a first-\/person perspective using raycasting.
\item {\bfseries{Textured Walls \& Floor Casting}}\+: Supports wall textures and basic floor and ceiling rendering.
\item {\bfseries{Player Movement}}\+: Includes rotation and movement with some collision detection.
\item {\bfseries{Configurable Map Parsing}}\+: Reads {\ttfamily .cub} files defining walls, textures, and player spawn.
